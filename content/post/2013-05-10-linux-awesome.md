How to use projector when use awesome
---
Just run cmd `$ xrandr --output VGA1 --auto`


Install Awesome
---
1. yum install -y awesome
2. copy .config/awesome/rc.lua
3. copy .xinit and .Xresources
4. yum install -y xorg-x11-xinit
